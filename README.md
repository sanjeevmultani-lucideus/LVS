# LVS
LVS is a VAPT Simulator which acts like Vulnerable Web Application covering all the Major Owasp Top 10 (2013) vulnerabilities. The web application is Based on HTML, PHP, JS, SQL and CSS. Any Web Application Newbie can use it to get a hands-on on most common web attacks like SQLI, XSS, CSRF, Invalidated Redirects etc. Kindly reach me out on LinkedIn for any issues.
